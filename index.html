    • Mobile Edge Computing based Task Offloading and Joint Multi-dimensional Resource Allocation in Heterogeneous Networks (National Natural Science Foundation of China) Demo 2019.01-Now,Participate

      • Responsible for the design of efficient user association and task offloading strategies for differentiated task requirements.
        Based on the precise characterization of terminal task attributes in heterogeneous MEC networks, we design an efficient offloading scheme for mobile terminals and multi-level edge computing servers, taking into account various influencing factors such as user location, user preferences, and multi-level server load and computing capacity.

    • Key Technology Research of Intelligent Vehicle Networking Based on Vehicle Behavior Analysis (National Natural Science Foundation of China International (Regional) Cooperation and Exchange Key Projects ) Demo 2019.01-Now,Participate

      • Responsible for data-driven intelligent resource management for heterogeneous vehicle networks.
        By using big data analysis and deep learning theory to deeply mine the historical communication data in Telematics, we can accurately predict the popularity of future contents in a specific time and space, and intelligently design caching strategies by combining the differentiation of communication service priorities, the number of feasible caching sites and mobility.
